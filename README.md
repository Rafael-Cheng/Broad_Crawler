## Introduction:

   This is a project aiming to crawl a variety of web pages(especially pages of news) with a spider, a.k.a. broad crawler.

## Features:
   
   The broad crawler should support the following features:

   * Title Extractor
   
   * URL Extractor
   
   * Date Extractor
   
   * Main Content Extractor
